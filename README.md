### Prerequisites

### Running the Demos
1. Clone this repository:
   ```bash
   git clone https://github.com/doribd/htmls.git
   ```

2. Navigate to the project directory:
   ```bash
   cd htmls
   ```

3. Open any HTML file in your web browser:
